
1.Name of my project:**KeenKeeper**


2.A short description:"**KeenKeeper** is a beautiful and intuitive web application designed to help users maintain meaningful relationships. It acts as a personal shelf for tracking friends, logging interactions, and nurturing connections that matter most.
Built with modern React and a clean UI, KeenKeeper makes relationship management simple, visual, and enjoyable"


3.Key features of your project :
 **Smart Friend Management**: Browse friends with status indicators (Overdue, Almost Due, On Track) and rich profile details
 **Interactive Timeline**: Log calls, texts, and video calls with one click and view your complete interaction history
 **Live Friendship Analytics**: Dynamic pie chart that updates in real-time as you log interactions (built with Recharts)
 **Fully Responsive Design**: Seamless experience across mobile, tablet, and desktop
 **Clean Navigation**: Modern navbar with active page highlighting and smooth mobile menu


 4.Technologies I used:
  **React.js** - Frontend library
  **React Router DOM** - Client-side routing
  **Tailwind CSS** - Utility-first styling
  **Recharts** - Data visualization (Pie Chart)
  **Lucide React** - Beautiful icons
  **React Context API** - State management for interactions
